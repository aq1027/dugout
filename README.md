# ⚾ Dugout

An offline-first baseball and softball scorekeeper PWA. Score games play-by-play, track stats, and never lose a game record.

**Status:** Early development — Phase 1 (project setup & data model) complete.

## Tech Stack

- React + TypeScript + Vite
- PWA (vite-plugin-pwa) — installable, works offline
- IndexedDB (Dexie.js) — local persistence, no backend
- Event-sourced game engine — full undo/redo, play-by-play history

## Getting Started

```sh
npm install
npm run dev
```
