# NCAA Softball Rules Reference

## Game Structure
- **Innings:** 7 (regulation)
- **Players per side:** 9 (or 10 with DP/FLEX)
- **Extra innings:** Continue until a winner is decided
- **Extra-inning auto runner:** Yes — runner placed on 2B (international tiebreaker rule)

## Roster & Lineup
- **Designated Hitter (DH):** Yes
- **DP/FLEX:** Yes — allows a 10th player (Designated Player) with a FLEX defensive replacement
- **Everyone bats:** No (unless using DP/FLEX with 10-player batting order)

## Run Rules
- **Mercy rule:** 8-run lead after 5 complete innings
- **Walk-off:** Yes

## Mound Visits
- No hard limit; umpire discretion for pace of play

## Pitching
- Underhand delivery
- No pitch clock currently enforced
