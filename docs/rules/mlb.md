# MLB Rules Reference

## Game Structure
- **Innings:** 9 (regulation)
- **Players per side:** 9
- **Extra innings:** Continue until a winner is decided
- **Extra-inning auto runner:** Yes — runner placed on 2B to start each extra half-inning (since 2020, made permanent 2023)

## Roster & Lineup
- **Designated Hitter (DH):** Yes (universal DH since 2022)
- **DP/FLEX:** No
- **Everyone bats:** No — only 9 batting slots

## Run Rules
- **Mercy rule:** None
- **Walk-off:** Yes — home team wins immediately when taking the lead in the bottom of the 9th or later

## Mound Visits
- **Limit:** 5 per team per 9-inning game (1 additional per extra inning)
- **Pitching changes do not count** toward the limit

## Pitch Clock (since 2023)
- **Bases empty:** 15 seconds
- **Runners on:** 18 seconds
- **Batter must be ready:** 8 seconds before timer expires
- **Violation (pitcher):** Automatic ball
- **Violation (batter):** Automatic strike
