# Custom Rules in Dugout

When you select "Custom" as your league preset (or modify any preset's defaults), you have full control over all game rules.

## Configurable Fields

| Field | Description | Range |
|-------|-------------|-------|
| **Innings** | Number of regulation innings | 1–15 |
| **Players per side** | How many bat in the lineup | 8–12 |
| **Designated Hitter** | Allow a DH in the lineup | On/Off |
| **DP/FLEX** | Allow a designated player + flex (softball) | On/Off |
| **Extra-inning auto runner** | Place a runner on 2B at the start of each extra inning half | On/Off |
| **Everyone bats** | Use continuous batting order (all rostered players bat) | On/Off |
| **Mercy rule** | Run differential to end the game early | 0 = off, or 1–30 |
| **Mercy after inning** | Mercy rule applies after this inning is complete | 1–15 |

## How Presets Work

1. Select a league preset (e.g., MLB, NCAA, HS, Little League)
2. All rule fields are auto-filled with that league's defaults
3. Click "Customize Rules" to reveal the full editor and adjust individual fields
4. Once any field is changed, the preset switches to "Custom"

## Tips

- **Rec league?** Start with the closest preset and adjust innings/mercy as needed.
- **Tournament?** Many tournaments use modified mercy rules — pick the base level and tweak.
- **Not sure?** MLB or NCAA defaults cover most standard games.
