# High School Softball Rules Reference

> Note: Rules governed by NFHS. State associations may adopt modifications.

## Game Structure
- **Innings:** 7 (regulation)
- **Players per side:** 9
- **Extra innings:** Continue until a winner is decided
- **Extra-inning auto runner:** No (not standard under NFHS)

## Roster & Lineup
- **Designated Hitter (DH):** Optional
- **DP/FLEX:** Yes (NFHS adopted DP/FLEX)
- **Everyone bats:** No (some states allow continuous batting order)

## Run Rules
- **Mercy rule:** 12-run lead after 3 complete innings; 10-run lead after 5 complete innings
- **Walk-off:** Yes

## Mound Visits
- No set limit; umpire manages pace

## Pitching
- Underhand delivery
- No pitch count limits (innings/day limits vary by state)
