# High School Baseball Rules Reference

> Note: Rules governed by NFHS (National Federation of State High School Associations). State associations may adopt modifications.

## Game Structure
- **Innings:** 7 (regulation)
- **Players per side:** 9
- **Extra innings:** Continue until a winner is decided
- **Extra-inning auto runner:** No (not standard under NFHS; some states may adopt)

## Roster & Lineup
- **Designated Hitter (DH):** Optional (NFHS rules allow it)
- **DP/FLEX:** No
- **Everyone bats:** No (some states allow continuous batting order)

## Run Rules
- **Mercy rule:** 10-run lead after 5 complete innings (or 4½ if home team leads)
- **Walk-off:** Yes

## Mound Visits
- No set limit under NFHS rules; umpire may restrict for pace

## Pitching
- **Pitch count limits:** Vary by state (typically 105-110 pitches per day with mandatory rest)
- No pitch clock
